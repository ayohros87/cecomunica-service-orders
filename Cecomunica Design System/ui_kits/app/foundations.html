Cómo usar

Envuelve la tabla en .responsive-table-wrap y duplica los mismos datos como una lista de .responsive-card dentro de .responsive-cards. El CSS hace el swap automático en @media (max-width: 900px). En producción suele renderizarse una sola vez vía JS y el snippet HTML elige la representación.
