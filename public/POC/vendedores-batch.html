Esta herramienta prepara los equipos de un cliente y genera un archivo — no los carga al sistema. Al terminar, descarga el JSON y envíalo a recepción para que los cargue.
1

Datos del cliente

Pendiente
Escribe parte del nombre y selecciónalo de la lista — sus grupos se cargan solos.
2

Configuración del lote

Pendiente
Para crear un grupo nuevo, ve a Administrar grupos (menú “Más” en POC).
Un nombre por línea — uno por cada equipo. El serial y el Unit ID se asignan después, cuando recepción los carga.
3

Equipos

Sin filas
Toca los grupos de cada equipo en la columna Grupos. Usa “Aplicar a todas” para marcar un grupo en todas las filas a la vez.
4

Revisión y exportación

Pendiente
Envía el JSON a recepción para cargar los equipos. El Excel es solo para revisar.
