🔄 Migración: cliente_nombre_lower

¿Qué hace esta migración?

Agrega el campo cliente_nombre_lower a todos los contratos existentes para habilitar la búsqueda en toda la base de datos.

Este campo convierte el nombre del cliente a minúsculas (ej: "Juan Pérez" → "juan pérez") para permitir búsquedas sin distinción de mayúsculas.
