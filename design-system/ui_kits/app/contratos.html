Contrato · Impresión

Plantilla de impresión A4 para contratos. Usa las mismas clases .print-* que print-demo.html (órdenes), con .print-sig-grid.cols-3 para las tres firmas requeridas en contratos (cliente, Cecomunica, vendedor) y un quiebre de página .page-break-before para los términos y condiciones. Se promueven a public/css/print-base.css para que imprimir-contrato.html las use directamente.

Abrir plantilla de impresión Abre print-demo-contrato.html en una pestaña nueva — usar el botón ↗ Abrir vista de impresión dentro para probar el resultado real.
Clases promovidas a print-base.css
  • .print-header, .print-header-logo, .print-header-brand, .print-header-doc-type — encabezado con logo + datos de empresa + badge de tipo
  • .print-doc-title — título del documento (uppercase, navy, centered)
  • .print-summary-row, .print-summary-id, .print-chips, .print-chip — fila resumen con ID + chips de estado
  • .print-section-title — separador de secciones (navy, uppercase)
  • .print-grid, .print-field (con dt/dd) — pares clave/valor en cuadrícula
  • .print-table — tabla de líneas con tfoot para subtotal/ITBMS/Total
  • .print-obs-totals, .print-obs — bloque de observaciones + totales
  • .print-sig-grid + .cols-3 — filas de firmas (2 o 3 columnas)
  • .print-sig-block, .print-sig-space, .print-sig-label, .print-sig-name, .print-sig-date, .print-sig-electronic
  • .print-verification, .print-verification-qr, .print-verification-body — bloque QR + códigos de verificación
  • .print-terms-title, .print-terms-body — términos en página aparte (dos columnas, justificado)
  • .print-acceptance — bloque de aceptación final con firma única
  • .print-footer-note — texto legal/branding al pie
  • .keep-together, .page-break-before, .page-break-after — utilidades de paginación
